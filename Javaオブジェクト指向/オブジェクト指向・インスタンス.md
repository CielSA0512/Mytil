## 〇オブジェクト指向とは
- 「ある役割を持ったモノ」ごとにクラスを分割し、モノとモノとの関係性を定義していくことでシステムを作り上げようとするシステム構成の考え方。
- 物を表現するためには「属性（どのようなものか」「機能（何ができるか）」が必要

### フィールド・メンバ
- クラスブロック直下で定義された変数のことを「フィールド」といい、そのクラスの「属性」を表す。
- メソッドはそのクラスの「機能」を表す。
- メソッドとフィールドを合わせて「メンバ」と言う　( 下の図の青枠 )
