## 論理演算子
- かつ・・・「&&」 ※両方trueでなければ全体はtrueにならない
- または・・・「||」 ※片方がtrueであれば全体もtrue
- ～でない・・・「!」 ※条件の一番前につける
- Notイコール「^」 ※両方ともfalseのとき全体はtrueになる
